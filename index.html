Site simples para demonstrar como um cookie de sessão pode ser criado e lido.

Notes

• This is a safe simulation: the "suspicious" script only logs to the console and does NOT send data anywhere. Use for teaching/demonstration.

• Open browser DevTools (Console) to see logs when you click the buttons.

How to use:

  1. Fill username/password → click Login.
  2. Open DevTools → Console.
  3. Click Simulate suspicious script to show obfuscated reader (it will log the token).
  4. Explain risks: never paste scripts you don't understand.
